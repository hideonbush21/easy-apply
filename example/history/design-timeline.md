# 留学申请时间轴 - 设计文档

## Part 1: 视觉设计系统

### 色彩系统

**主色调：**
- 主色: `#1dd3b0` (青绿色)
- 次色: `#0a0a0a` (近黑色)
- 白色: `#ffffff`

**辅助色：**
- 曲线基线: `#E5E7EB` (gray-200)
- 已完成段: `#9CA3AF` (gray-400)
- 激活高亮: `#1dd3b0` (主色)
- 节点当前: `#1dd3b0`
- 节点未来: `#E5E7EB`
- 卡片背景: `#ffffff`
- 遮罩层: `rgba(0, 0, 0, 0.4)`

### 字体系统

**字体家族：** Inter, system-ui, sans-serif

**字号规范：**
- 节点月份: 14px, font-weight: 600
- 卡片标题: 16px, font-weight: 700
- To Do 项: 14px, font-weight: 400
- 进度文字: 12px

### 间距系统

- 卡片宽度: 280px
- 卡片圆角: 12px
- 节点尺寸: 12px(已完成) / 16px(未来) / 24px(当前)
- 卡片阴影: `0 10px 40px rgba(0,0,0,0.1)`

## Part 2: 全局动画与交互

### 页面加载动画
- 曲线绘制动画: stroke-dashoffset 从 100% 到 0%
- 节点依次淡入: stagger 100ms
- 卡片上浮进入: translateY 20px → 0, opacity 0 → 1

### 滚动行为
- 水平滚动: 鼠标滚轮映射为横向滚动
- 滚动惯性: CSS scroll-behavior: smooth
- 节点吸附: 滚动停止时最近节点对齐视口中心

### 悬浮效果
- 节点: translateY(-12px) scale(1.1), 300ms, cubic-bezier(0.34, 1.56, 0.64, 1)
- 卡片: translateY(-4px), 阴影加深
- 曲线高亮: 当前段变为主色，宽度 4px，加发光

### 二级界面动画
- FLIP 动画: 卡片放大填满屏幕
- 遮罩淡入: opacity 0 → 1, 300ms
- 内容滑入: translateY 20px → 0

## Part 3: 内容区块设计

### Section: 时间轴主界面

**布局：**
- 全屏容器 100vw × 100vh
- 水平滚动层 width: 300vw
- SVG 曲线覆盖整个滚动区域
- 节点和卡片绝对定位在曲线之上

**曲线设计：**
- 三次贝塞尔曲线连续拼接
- 4段曲线形成由远及近的透视感
- 线宽: 3px (基线), 4px (高亮)

**5个时间节点：**

1. **1-3月 申请冲刺期**
   - 位置: 曲线起点附近
   - To Do: 补齐材料换uncon, 确认押金Deposit, 换取CAS, 缴纳押金激活IT账户

2. **4-5月 签证准备期**
   - 位置: 曲线中段
   - To Do: 准备签证材料, 预约签证, 肺结核检查, 存款证明

3. **6-7月 行前准备期**
   - 位置: 曲线中后段
   - To Do: 预订机票, 安排住宿, 购买保险, 准备行李清单

4. **8-9月 开学前期**
   - 位置: 曲线后段
   - To Do: 注册报到, 领取BRP, 银行开户, 熟悉校园

5. **抵达英国后**
   - 位置: 曲线终点
   - To Do: 参加新生周, 办理手机卡, 注册GP, 适应生活

**交互：**
- 点击节点/卡片 → 展开二级详情页
- 悬浮节点 → 高亮曲线段 + 卡片上浮

---

### Section: 二级详情页

**布局：**
- 全屏遮罩: rgba(0, 0, 0, 0.4)
- 内容卡片: 居中, max-width 640px, max-height 90vh
- 顶部栏: 返回按钮 + 月份标签 + 关闭按钮

**内容：**
- 环形进度指示器 (SVG)
- To Do 列表 (可勾选)
- 文件上传区域
- 添加自定义待办

**交互：**
- 勾选动画: 对勾绘制 + 项右移回弹
- 文件上传: 拖拽 + 点击选择
- 点击遮罩/关闭 → 返回主界面
